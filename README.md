# ETAP Expert - Guia de Decisão Industrial

Aplicativo PWA focado no comissionamento e diagnóstico do módulo **1783-ETAP Allen-Bradley**.

### 🛠 Funcionalidades
- Árvore de decisão para IP conhecido/desconhecido.
- Passo a passo para ferramenta **BootP DHCP Tool**.
- Consulta rápida de **Switches Rotativos**.
- Diagnóstico via **Web Server** e **RSLinx**.

### 📲 Instalação
Acesse pelo link do GitHub Pages e clique em **"Adicionar à tela de início"** no seu smartphone.
